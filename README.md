# 🧩 Паттерны проектирования  GoF (Gang of Four)

## 📌 О проекте

Этот проект представляет собой коллекцию наглядных примеров реализации всех 23 паттернов проектирования, описанных в культовой книге "Банды четырёх" (GoF). Каждый паттерн сопровождается:

- ✅ Кратким теоретическим описанием
- ✅ Примером использования на C#
- ✅ Демонстрацией работы с выводом в консоль

Проект может использоваться как:
- **Учебное пособие** для изучения паттернов
- **Шпаргалка** для быстрого освежения знаний
- **Стартовая точка** для реализации паттернов в собственных проектах

## 🚀 Быстрый старт

### Требования
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) или выше

### Установка и запуск

```bash
# Клонируем репозиторий
git clone git@github.com:burstCode/PatternsCourse.git
cd patternscource

# Собираем проект
dotnet build

# Запускаем (интерактивный режим)
dotnet run

# Или запускаем конкретный паттерн
dotnet run -- --singleton
